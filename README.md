# vue-ui-components
Useful components for websites that use full vue.js power.

---

It will include components:
- banner;
- accordion;
- modal;
- slider;
- gallery;
- form;
- buttons;
- inputs;
- table;
- header;
- footer.



## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
